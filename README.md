# energy-center-investment-model
# 能源中心投资与能耗测算工具集

## 一、项目背景
本项目用于集中供热/供冷能源中心前期方案比选，
适用于住宅、办公、园区类项目的技术经济测算。

## 二、适用范围
- 供热方式：热泵 / 锅炉 / 组合系统
- 供冷方式：冷水机组 / 热泵制冷
- 不包含：生活热水（可后续扩展）

## 三、主要功能
- 项目总投资估算
- 年运行能耗及费用测算
- 静态投资回收期
- 现金流分析（逐步完善）

## 四、目录说明
- investment-model：投资测算模型
- hvac-calculation：能耗与系统计算
- standards-review：标准/规范校对记录
- app：Python 小程序（后续）

## 五、使用说明
1. Excel 文件可直接打开使用
2. Python 程序需安装 Python 3.10+
3. 所有参数均可根据项目实际调整

## 六、版本记录
- v1：基础投资测算模型
- v2：增加运行费用与现金流
- v3：增加融资与可视化（规划中）
